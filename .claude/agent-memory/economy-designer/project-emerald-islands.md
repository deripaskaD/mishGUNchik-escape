Изумрудные Острова — 2D аниме-RPG на Godot 4.7 / GDScript. Brownfield-проект: игра уже реализована.

**Why:** Контекст для всех будущих экономических решений.

**How to apply:** Всегда читать canon.md и game_data.gd перед проектированием любой системы.

## Ключевые факты

- Level cap: 60 (старт), сезонные повышения +5 каждые 3 месяца
- 4 класса: Наёмник, Дворянин, Святая, Следопыт — по 2 ветки специализации (ур. 20)
- 8 зон: Изумрудный → Пески → Ледяной → Вулканический → Руины → Топи → Коралловые рифы → Бездна
- Двойная прогрессия: суша (XP) + море (Sea XP отдельно, морской уровень 1→30)
- 5 фракций: Гильдия Мореходов, Орден Эфира, Вольные Торговцы, Стражи Островов, Пираты Бездны
- Единственная валюта: золото
- 5 редкостей: Обычный/Необычный/Редкий/Эпический/Легендарный (мультипликаторы 1.0/1.65/2.4/3.4/4.7)
- 3 слота снаряжения: weapon, armor, ring (код в game_data.gd)
- Инвентарь: 24 слота
- Монетизация-нейтрально: никакого pay-to-win

## Реализованные системы (код готов)

- XP-формула: `48 × l^1.62`
- Формула цены: `70 × (t+1)^1.6`
- sell_value: `(item_power + 4) × (rarity+1) × 0.8`
- 7 талантов × 8 рангов (might/vit/focus/fortune/fury/ward/swift)
- 3 умения на класс, 7 рангов каждое
- Система дропа и редкости (roll_rarity по весам)
- Fortune talent: gain_mul = 1.0 + rank × 0.08
