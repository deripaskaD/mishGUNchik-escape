Пользователь — creative director проекта. Даёт полные задания с контекстом (ссылки на файлы + список того, что нужно создать). Ожидает конкретные, реализуемые спецификации — не абстрактные рекомендации.

**Стиль подачи**: русский язык, конкретные числа в dp/sp, привязка к Godot-нодам и существующим скриптам. Формат — Markdown-таблицы и списки, не сплошной текст.

**Ограничения роли**: не трогать код, не принимать визуальных решений за арт-директора, писать только в design/ux/.

**Паттерн работы**: пользователь даёт «сделай всё сразу» задание с полным контекстом. Auto Mode активен — действовать без лишних уточнений, когда контекст достаточен.
