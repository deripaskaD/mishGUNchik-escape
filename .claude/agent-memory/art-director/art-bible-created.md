Арт-библия v1.0 написана и размещена в `design/art/art-bible.md`.

**Why:** Стиль игры — аниме, яркий, солнечный, НЕ мрачный (прямо из canon.md). Референс Florensia. Top-down mobile 2D RPG.

**How to apply:** Все будущие спрайты, VFX, UI и тайлы проверять против этого документа. Промпты для gen_sprites.py берутся из секции 9 арт-библии.

Ключевые решения:
- 5 визуальных столпов (Anime Cel-Shaded / Яркое / Читаемый Силуэт / Приключенческий / Морской)
- Палитры для 8 биомов с hex-кодами
- Цветовые коды 4 классов: merc=красный/стальной, noble=синий/золотой, saint=белый/золотой/розовый, expl=зелёный/коричневый
- Фракции закодированы цветом и символом
- UI-семантика цветов (HP=алый, мана=синий, XP=фиолетовый, золото=золотой)
- Единый стиль-префикс промпта для Flux
