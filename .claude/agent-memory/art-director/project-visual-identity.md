**Стиль:** Anime cel-shaded, яркое/солнечное, mobile top-down 2D RPG. Референс: Florensia, Ragnarok Online, MapleStory.

**Why:** Канон мира (`design/world/canon.md`) явно задаёт «светлое аниме-приключение», НЕ мрачный стиль. Don't Starve эстетика — запрещена.

**How to apply:** При любом арт-решении проверять: высокая насыщенность? читается силуэт? не слишком реалистично?

5 столпов:
1. Anime Cel-Shaded — чёткие контуры, плоские заливки 1–2 ступени тени
2. Яркое & Солнечное — S 60–90%, V 70–95%, тёплый свет
3. Читаемый Силуэт — узнаётся с 50 px на мобиле
4. Приключенческий Вайб — округлые формы, преувеличенные детали
5. Морская Романтика — Лазурь живая и сияющая

Запрещено: десатурация, реалистичные тени, перегруженный силуэт, тёмный фон без акцентов (кроме зоны Бездна — единственное исключение).
