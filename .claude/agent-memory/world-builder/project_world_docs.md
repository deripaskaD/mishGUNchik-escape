Созданы три продакшн-лор-документа в `design/world/`:

1. **world-bible.md** — космология эфира (три состояния), история (Прима → Раскол → Архипелаг), Бездна как сущность-паразит, хронологическая линия от Изначальных до наших дней, пантеон, язык/культура, экология.

2. **factions.md** — 5 фракций: Гильдия Мореходов (Элара Стурм), Орден Эфира (Зоран + Ниа Ферн), Вольные Торговцы (Фабио Лант), Стражи Островов (Маршал Каэн), Пираты Бездны (Кэрос). У каждой: 5 уровней репутации с наградами, отношения между фракциями, связь с эндгеймом. Пираты — обратный антирейтинг «Слава Охотника».

3. **zones.md** — 8 зон + Серые Острова + морские хайвеи. Осколки Эфира 1–7 распределены по зонам 1–7, 8-й в Зоне Бездны. Три ключа к финальному рейду у Гильдии / Стражей / Ордена (уровень Легенды).

**Why:** первичный мировой лор для обеспечения нарративной согласованности между всеми агентами.

**How to apply:** при создании новых лор-элементов (квесты, NPC, данжи) сверяться с этими тремя документами. canon.md остаётся первичным seed-документом, world-bible.md — его расширением.
