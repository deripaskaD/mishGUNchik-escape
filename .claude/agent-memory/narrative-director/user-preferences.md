**Масштаб**: пользователь явно указал «десятки часов + эндгейм, НЕ 30-мин прототип». Всегда мыслить продакшн-масштабом.

**Язык**: русский во всех нарративных документах.

**Формат**: структурированные таблицы для каталогов (квесты, NPC). Markdown с заголовками.

**Соответствие канону**: строго соблюдать design/world/canon.md. Любые новые нарративные элементы — проверять против канона.

**Тон**: светлое аниме-приключение (Florensia, не тёмное фэнтези). Потери временные, победы настоящие.

**Why:** пользователь недоволен «рамочным» нарративом из 29 квестов — хочет полноценный продакшн-уровень.

**How to apply:** при любом нарративном запросе сразу ориентироваться на полный масштаб без дополнительных уточнений о «глубине».
