# Timokha Escape

Казуальный квест-комедия **с постоянной погоней** для iOS/Android (Godot 4.7 / GDScript).
Оммаж жанру «Побег от …»: шкодливый шалун **всё время гонится** за тобой по локации, а ты
на бегу выполняешь цепочку дурацких мини-игр-поручений (лепи пирожки, завари чай, наколи
дров) и в финале главы вырываешься на свободу. Все ассеты — оригинальные плейсхолдеры.

## Как запустить
1. Установи **Godot 4.7**.
2. Открой проект: `Import` → выбери `project.godot` в корне.
3. Нажми **Play (F5)**. Главная сцена — `scenes/main.tscn`.

Из консоли:
```
/путь/к/Godot --path . 
```

## Управление
- **Движение:** стрелки / перетаскивание пальцем (плавающий джойстик).
- **Мини-игры** (контекстные кнопки справа-внизу или клавиатура):
  - Пирожки — тапай (**ТАП** / Пробел).
  - Чай — удерживай и отпусти в зелёной зоне (**ДЕРЖИ** / Пробел).
  - Дрова — чередуй **← →** (кнопки / Z и X).
- **Esc** — пауза (Продолжить / Заново / В меню).

## Цель
Беги от шалуна. Встань в жёлтую зону (там он замедляется), выполни задачу. Когда все
задачи сделаны — появится **ВЫХОД**: добеги до него (шалун на полной скорости). Поймал —
не страшно: сброс прогресса задачи, отскок, без Game Over. Собирай 3 звезды за главу.

## Структура
```
scripts/
  data/game_data.gd        # GameData (autoload): главы, задачи, тюнинг
  core/game_state.gd       # GS (autoload): прогресс, сохранение
  core/world.gd            # геймплей главы (движение, погоня, мини-игры, побег)
  core/pursuer.gd          # логика преследователя
  core/quest_manager.gd    # цепочка задач
  core/minigame_base.gd    # базовый класс мини-игр
  core/minigames/          # pies / tea / wood
  audio/{sfx,music}.gd     # автозагрузки звука (толерантны к отсутствию ассетов)
  main.gd                  # меню / выбор глав / результаты
scenes/main.tscn           # точка входа
design/ docs/ production/  # GDD, архитектура/ADR, бэклог, QA (пайплайн CCGS)
tests/                     # headless-тесты логики и интеграции
art/                       # drop-in спрайты (art/<key>.png) — позже
```

## Звук и настройки
Процедурные плейсхолдер-SFX в `audio/sfx/` (`tools/gen_sfx.py`) + зацикленная фоновая
музыка `audio/music/theme.wav` (`tools/gen_music.py`). Всё заменяемо drop-in.
В меню есть **Настройки**: музыка вкл/выкл, звук вкл/выкл, сброс прогресса (сохраняются).
