# Art — drop-in спрайты

Рендер не зависит от наличия арта: если файла нет — рисуется fallback-фигура.
Чтобы заменить визуал, просто положи PNG по нужному ключу и переимпортируй проект.

## Манифест (ключи)

| Файл | Что это | Рекоменд. размер | Якорь |
|------|---------|------------------|-------|
| `player.png` | Игрок-беглец | 128×128, прозрачный фон | центр |
| `timokha.png` | Шалун-преследователь | 128×128, прозрачный фон | центр |

Спрайт рисуется вписанным в круг сущности (диаметр = 2×radius), центрирован.
Тинт фаз шалуна (замах/рывок) применяется как `modulate` — для нейтрального
спрайта это подсветка; если не нужно — игнорируй, логика не сломается.

## Плейсхолдеры
Текущие `player.png` / `timokha.png` сгенерированы `tools/gen_art.gd`
(`godot --headless --script res://tools/gen_art.gd`). Перегенерировать/заменить — свободно.

## Возможные будущие ключи (пока fallback-фигуры)
`bg_<chapter_id>.png` (фон главы), `hotspot.png`, `exit.png`, иконки мини-игр.
Добавление — по той же схеме: ключ + загрузка через автозагрузку `Art`.
