## [1.29.4] — 2026-07-01 — Перф: частицы на мобиле
### Performance
- Падающие листья (30) и искры костра (22) не были урезаны в `_mobile` (в отличие от светлячков/пыльцы/дождя/деревьев). Теперь: листья 12, искры 10 на телефоне (десктоп без изменений). Листья следуют за игроком → всегда в кадре, реальная экономия на мобильных.

## [1.29.3] — 2026-07-01 — Перф: sky-шейдер без лишнего прохода
### Performance
- Убран `render_mode use_quarter_res_pass` из sky-шейдера — он включал дополнительный quarter-res проход дорогого 4-октавного FBM каждый кадр, при этом `AT_QUARTER_RES_PASS` нигде не использовался (мёртвая работа GPU). Ambient берётся из radiance-кубмапы → не задет. День/ночь визуально идентичны (проверено). Помогает мобильным (жалобы на лаги).

## [1.29.2] — 2026-07-01 — Багфиксы (самопроверка)
### Fixed
- **Ночной «мельк» джампскейра** больше не проигрывает зацикленный `snd_dread` через `_play()` — это перезапускало ambient-дрон саспенса каждые 30–60 с (разрыв звука). Теперь мельк без звука (дрон играет непрерывно снизу).
- **Счётчик «Дневник (N/8)»** в паузе теперь учитывает уже найденные записки из сейва (был жёстко «0/8» до следующей находки у вернувшегося игрока).

## [1.26.0] — 2026-07-01 — Джампскейры (4 триггера)

## [1.25.0] — 2026-07-01 — Мягкий проигрыш: 3 жизни + быстрый респаун + воскрешение

### Notes

## [1.23.1] — 2026-07-01 — Небо больше не серое (fix fog_sky_affect) + ярче
### Fixed
- **Главная причина «плоского серого неба»**: `fog_sky_affect` по умолчанию = 1.0 → туман полностью вымывал небо в серость (днём) и съедал звёзды (ночью). Поставил `fog_sky_affect = 0.1` — небо осталось ярким, туман держит глубину леса. Теперь день — насыщенно-голубой с облаками, ночь — глубокий синий с россыпью звёзд.
### Changed
- Градиент дня сочнее (зенит `0.14,0.42,0.92`, горизонт светло-голубой `0.58,0.82,1.0` вместо серого).
- Облака объёмнее: двухслойный FBM + псевдо-объём (яркие макушки / голубоватый низ), пышные комки.
- День ярче: `tonemap_exposure` 1.06 днём → 0.85 ночью (по `nf`); `adjustment_saturation` 1.14→1.22; дневная дымка голубоватая.

## [1.23.0] — 2026-07-01 — Настоящее небо (sky-шейдер)
### Added
- **Кастомный sky-шейдер** (`SKY_SHADER`, `ShaderMaterial` на `Sky`) вместо плоского `ProceduralSkyMaterial`:

## [1.21.3] — 2026-06-29 — Фиксы веб-порта (управление + глифы)
### Fixed
- **Ходьба не работала в браузере.** На web `DisplayServer.is_touchscreen_available()` возвращает true даже на десктопе → игра включала мобильный тач-джойстик вместо WASD/мыши. Теперь `show_touch = _touch_flag or (is_touchscreen_available() and not OS.has_feature("web"))` — на web по умолчанию десктоп-управление (клавиатура+мышь, mouse-look по клику). Тач-события (`InputEventScreenTouch/Drag`) теперь обрабатываются только при `show_touch` (эмуляция touch-from-mouse не мешает). Проверено в браузере — WASD двигает, мышь крутит камеру.
- **Стрелка квеста и иконки рендерились квадратами (тофу).** Шрифт веб-экспорта не содержит Юникод-глифов (стрелки `↑→←↓↗↖`, эмодзи `🪵🌿`) — на десктопе спасал системный фолбэк, на web его нет. Компас переведён на ASCII-стрелки (`^ v < >`), эмодзи из статуса убраны. Теперь всё читается в браузере.
### Notes
- Изменения общие (десктоп тоже), на нативном десктопе вид чуть проще (ASCII-стрелки вместо Юникод). Нужно ПЕРЕЗАЛИТЬ обновлённый `build/mishGUNchik-escape-web.zip` на itch (заменить старый файл).

## [1.11.44] — 2026-06-27 — Фикс заиканий звука
### Fixed
- **«Залипание» звука (КОРЕНЬ)**: зацикленные звуки (ветер/дождь/сверчки) импортировались как QOA (compress/mode=2). В коде `loop_end = res.data.size()/2` верно только для PCM; для QOA `data` — сжатые байты (≈×8 меньше), поэтому loop_end указывал на ~0.4с от начала → луп крутил крошечный фрагмент = залипающий шум. ИСПРАВЛЕНО: дождь/ветер/сверчки переведены в PCM (compress/mode=0) + баковый loop_mode=1 → бесшовный полноразмерный луп.
- Профилактически: увеличен аудио-буфер (`output_latency` 15→40 мс) и снижены draw calls (деревья 760→320 и пр.) — оставлено как полезная мобильная оптимизация (туман скрывает разницу).
