# GDD — Pursuer AI (Шалун-преследователь)

*Status: Approved (solo) · 2026-06-24 · System #7*

## 1. Overview
Антагонист (шалун) **постоянно** преследует игрока по локации. Это определяющая
механика игры (по требованию пользователя — «как в референсе»). Погоня создаёт
непрерывное комичное давление, но остаётся прощающей и читаемой.

## 2. Player Fantasy
«За мной вечно носится этот шкодник — надо успеть сделать дело и удрать, смеясь».
Напряжение без страха (анти-пиллар *НЕ хоррор*).

## 3. Detailed Rules
- Шалун движется к игроку каждый кадр со скоростью `speed` (по умолчанию медленнее игрока).
- Периодически совершает **рывок** с явным телеграфом: фаза «замах» (почти замер, оранжевый, «!») → «рывок» (быстро, ярко-красный) → откат.
- Рывок не начинается, когда игрок в **замедляющей зоне** (у активной мини-игры) — там скорость × `slow_factor`.
- Касание игрока («осалил») при отсутствии оглушения → сетбэк: сброс прогресса текущей мини-игры, отскок игрока и шалуна, оглушение игрока `stun`, отмена рывка. **Нет Game Over.**

## 4. Formulas
- Позиция: `pos += (player - pos).normalized() * eff_speed * delta`.
- `eff_speed = speed` (walk) · `* 0.15` (telegraph) · `= dash_speed` (dash) · `* slow_factor` (в зоне).
- Рывок: каждые `lunge_interval` с, если `dist(player) < lunge_range` и не в зоне.
- Поимка: `dist(player) <= player_r + pursuer_r` и `stun_timer <= 0`.

## 5. Edge Cases
- Игрок в зоне при заходе → телеграф/рывок подавлены, но шалун всё ещё медленно подходит и может осалить (нельзя сидеть вечно).
- Двойная поимка подряд: после осаливания `stun` даёт грейс — проверка поимки выключена, пока идёт оглушение.
- Победа (`STATE_WIN`) → скорость 0 (шалун замирает).

## 6. Dependencies
Player Movement & Camera, Location & Bounds. Сигналит в Juice/Feedback и HUD через GS (`caught`).

## 7. Tuning Knobs
`speed` (per-chapter), `dash_speed`, `telegraph_time`, `dash_time`, `lunge_interval`,
`lunge_range`, `slow_factor`, `stun`, knockback. Все — в `GameData`/`Pursuer`.

## 8. Acceptance Criteria
- [ ] Шалун виден и преследует непрерывно во всех фазах, кроме победы.
- [ ] Рывок всегда телеграфируется (видимый «замах» ≥ `telegraph_time`).
- [ ] У зоны мини-игры шалун ощутимо замедляется.
- [ ] Поимка не даёт Game Over: сетбэк + отскок + оглушение, игра продолжается.
- [ ] Тест логики поимки/шага проходит (`tests/run_tests.gd`).
